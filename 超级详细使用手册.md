# 🚀 健康自检系统 - 超详细使用手册

## 📱 这是什么？

这是一个**健康管理系统**，有**两个部分**：

1. **用户端（小程序）** - 给**患者**使用，做健康自检、调理方案
2. **管理后台** - 给**医生/管理员**使用，查看和管理患者数据

### ❓ Q4：健康评分是什么意思？

**A：健康评分说明：**
```
┌──────────┬────────────────────────┐
│  分数    │         含义           │
├──────────┼────────────────────────┤
│ 80-100分 │ 优秀（健康状态良好）   │
│ 60-79分  │ 良好（基本健康）       │
│ 40-59分  │ 一般（需要注意调理）   │

└──────────┴────────────────────────┘
```

---

### ❓ Q5：BMI是什么意思？

**A：BMI（身体质量指数）说明：**
```
┌──────────┬────────────────────────┐
│  BMI值   │         含义           │
├──────────┼────────────────────────┤
│  <18.5   │ 体重过轻               │
│ 18.5-24  │ 体重正常 ✅            │
│  24-28   │ 超重                   │
│  >28     │ 肥胖                   │
└──────────┴────────────────────────┘

计算公式：BMI = 体重(kg) ÷ 身高²(m)
```

---

### ❓ Q6：血压多少算正常？

**A：血压标准说明：**
```
┌──────────────┬────────────────────────┐
│  血压值      │         含义           │
├──────────────┼────────────────────────┤
│ <120/80      │ 正常 ✅                │
│ 120-130/80   │ 正常高值               │
│ 130-140/80-90│ 高血压1期              │
│ >140/90      │ 高血压2期              │
└──────────────┴────────────────────────┘

第一个数字是收缩压（高压）
第二个数字是舒张压（低压）
```
