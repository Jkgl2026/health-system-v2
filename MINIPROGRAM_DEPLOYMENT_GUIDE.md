# 微信小程序提交审核发布完整指南

本文档将指导您如何将健康自检小程序提交到微信小程序平台进行审核和发布。

---

## 📋 准备工作清单

### 必需材料
- [ ] 微信账号（已认证的个人或企业账号）
- [ ] 小程序名称
- [ ] 小程序简介
- [ ] 小程序图标（108x108像素，不超过500KB）
- [ ] 微信支付商户号（如果需要支付功能）
- [ ] 服务器域名和SSL证书

### 开发工具
- [ ] 微信开发者工具（最新版本）
- [ ] 小程序代码（miniprogram-code目录）
- [ ] AppID（微信小程序唯一标识）

---

## 🚀 第一步：注册微信小程序

### 1.1 访问微信公众平台

1. 打开浏览器，访问：https://mp.weixin.qq.com
2. 点击右上角的「立即注册」
3. 选择「小程序」

### 1.2 填写注册信息

1. **邮箱注册**
   - 使用未注册过微信平台账号的邮箱
   - 填写邮箱地址
   - 获取并填写验证码

2. **设置密码**
   - 设置登录密码
   - 确认密码

3. **选择主体类型**
   - **个人**：适合个人开发者，功能受限（不支持支付、微信登录等）
   - **企业**：适合公司，功能完整（支持支付、微信登录、分享等）

### 1.3 邮箱激活

1. 登录注册邮箱
2. 点击微信发送的激活链接
3. 激活成功后跳转到登录页面

### 1.4 登录并完善信息

1. 使用注册的邮箱和密码登录
2. 扫描二维码绑定微信账号（必须是微信账号本人操作）
3. 完善账号信息：
   - 主体信息（个人/企业）
   - 管理员信息
   - 联系方式

### 1.5 获取AppID

注册完成后，系统会分配一个唯一的小程序AppID，格式类似：
```
wx1234567890abcdef
```

**重要：** 记录下这个AppID，后续开发需要使用。

---

## 💻 第二步：安装微信开发者工具

### 2.1 下载开发者工具

访问微信开发者工具官网：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

根据操作系统下载对应版本：
- Windows 64位
- Windows 32位
- macOS

### 2.2 安装开发者工具

1. 运行下载的安装程序
2. 按照提示完成安装
3. 安装完成后，启动微信开发者工具
4. 使用微信扫码登录

---

## 📁 第三步：导入小程序代码

### 3.1 创建小程序项目

1. 打开微信开发者工具
2. 点击「+」号创建新项目
3. 填写项目信息：
   - **项目名称**：健康自检管理
   - **目录**：选择 `miniprogram-code` 文件夹
   - **AppID**：填入第一步获取的AppID
   - **开发模式**：小程序
   - **后端服务**：不使用云服务（使用自己的服务器）

4. 点击「创建」

### 3.2 配置AppID

打开 `miniprogram-code/app.config.ts` 文件，修改AppID：

```typescript
export default {
  appId: 'wx1234567890abcdef', // 替换为您的AppID
  // ...其他配置
} as App.Config;
```

### 3.3 配置服务器域名

如果您的健康自检应用使用了自己的服务器，需要在微信公众平台配置服务器域名：

1. 登录微信公众平台
2. 进入「开发」→「开发管理」→「开发设置」
3. 找到「服务器域名」
4. 配置以下域名：
   - **request合法域名**：您的API域名
   - **socket合法域名**：WebSocket域名（如果使用）
   - **uploadFile合法域名**：文件上传域名
   - **downloadFile合法域名**：文件下载域名

**注意：** 所有域名必须使用HTTPS，并且需要配置有效的SSL证书。

---

## 🔧 第四步：配置小程序信息

### 4.1 设置小程序基本信息

1. 登录微信公众平台
2. 进入「设置」→「基本设置」
3. 填写以下信息：
   - **小程序名称**：健康自检管理（必须唯一）
   - **简介**：通过科学的症状自检和健康要素分析，了解身体状况
   - **头像**：上传108x108像素的图标
   - **类目**：选择合适的类目（如：医疗 > 医疗信息）

### 4.2 配置服务类目

1. 进入「设置」→「基本设置」→「服务类目」
2. 点击「添加」
3. 选择服务类目：
   - 主要类目：医疗 > 医疗信息
   - 备用类目：工具 > 其他

**注意：** 某些类目需要提交资质证明。

### 4.3 隐私设置

1. 进入「设置」→「隐私设置」
2. 配置用户信息保护指引
3. 说明小程序如何收集和使用用户数据

---

## 🧪 第五步：本地测试

### 5.1 真机调试

1. 在微信开发者工具中，点击「真机调试」
2. 使用微信扫描二维码
3. 在手机微信中打开小程序进行测试
4. 测试所有功能：
   - 用户注册/登录
   - 症状自检
   - 健康分析
   - 数据保存
   - 个人中心

### 5.2 性能测试

1. 检查小程序加载速度
2. 测试网络不稳定情况下的表现
3. 检查内存使用情况
4. 确保没有明显卡顿

### 5.3 兼容性测试

在不同设备和系统版本上测试：
- iOS 13+
- Android 7.0+
- 不同屏幕尺寸（4.7英寸、5.5英寸、6.5英寸等）

---

## 📦 第六步：上传代码

### 6.1 上传到微信服务器

1. 在微信开发者工具中，点击「上传」
2. 填写版本号：
   - **版本号**：1.0.0
   - **项目备注**：首次提交，包含用户端和管理员端完整功能

3. 点击「上传」

### 6.2 等待上传完成

上传完成后，会显示上传成功提示，包含：
- 版本号
- 提交时间
- 项目备注

---

## ✅ 第七步：提交审核

### 7.1 进入提交审核页面

1. 登录微信公众平台
2. 进入「管理」→「版本管理」
3. 找到刚才上传的版本（1.0.0）
4. 点击「提交审核」

### 7.2 填写审核信息

#### 页面信息
1. **功能页面**：
   - 首页：用户自检入口
   - 症状自检：选择症状进行健康评估
   - 自检结果：查看健康得分和建议
   - 健康分析：深度健康分析报告
   - 个人中心：用户数据管理

2. **页面截图**：
   - 为每个页面上传截图（尺寸：800x1200像素）
   - 确保截图真实反映小程序功能

#### 类目信息
1. **服务类目**：
   - 选择：医疗 > 医疗信息

#### 资质信息
1. 如果选择了需要资质的类目，上传相应的资质证明

### 7.3 提交审核

填写完所有信息后，点击「提交审核」

### 7.4 审核状态

提交审核后，小程序会进入以下状态：

1. **审核中**：1-7个工作日
   - 微信团队会审核小程序是否符合规范
   - 检查功能是否完整
   - 检查是否有违规内容

2. **审核通过**：可以发布
   - 可以选择「全量发布」或「分阶段发布」
   - 全量发布：所有用户都能使用
   - 分阶段发布：先给部分用户使用，逐步扩大范围

3. **审核驳回**：需要修改后重新提交
   - 查看驳回原因
   - 修改相关问题
   - 重新提交审核

---

## 🌟 第八步：发布上线

### 8.1 发布小程序

1. 审核通过后，在「版本管理」页面
2. 点击「发布」
3. 选择发布方式：
   - **全量发布**：立即对所有用户可见
   - **灰度发布**：先给10%-50%用户使用，观察稳定性

4. 填写发布说明
5. 点击「确定发布」

### 8.2 验证发布

1. 在微信中搜索小程序名称
2. 找到并打开小程序
3. 测试核心功能
4. 确认一切正常

---

## 📊 第九步：运营维护

### 9.1 数据统计

1. 进入「统计」→「数据分析」
2. 查看用户数据：
   - 日活/月活用户数
   - 用户留存率
   - 页面访问量
   - 功能使用情况

### 9.2 用户反馈

1. 进入「客服」→「客服消息」
2. 查看用户反馈
3. 及时回复和处理问题

### 9.3 版本更新

当需要更新功能时：

1. 修改代码
2. 本地测试
3. 上传新版本（版本号递增）
4. 提交审核
5. 审核通过后发布

---

## ⚠️ 重要注意事项

### 审核常见问题

1. **功能不完整**
   - 确保所有页面都能正常访问
   - 避免空页面或"开发中"页面

2. **内容违规**
   - 不涉及医疗诊断和处方
   - 不发布虚假健康信息
   - 不涉及赌博、色情等内容

3. **隐私问题**
   - 明确告知用户数据收集方式
   - 遵守《微信小程序平台运营规范》
   - 不收集不必要的用户信息

4. **用户体验**
   - 页面加载速度要快
   - 避免明显的卡顿
   - 适配各种屏幕尺寸

### 个人账号限制

个人账号的小程序有以下限制：
- ❌ 不能使用微信支付
- ❌ 不能使用微信登录
- ❌ 不能使用企业能力
- ✅ 可以使用基本功能
- ✅ 可以展示信息
- ✅ 可以收集用户反馈

如果需要完整功能，建议使用企业账号注册。

### 服务器要求

1. **HTTPS强制**
   - 所有接口必须使用HTTPS
   - 必须配置有效的SSL证书
   - 可以使用免费SSL证书（Let's Encrypt）

2. **域名备案**
   - 如果服务器在中国大陆，域名需要备案
   - 个人账号和企业账号都需要域名备案

3. **性能要求**
   - 接口响应时间不超过3秒
   - 支持高并发访问
   - 有足够的带宽和存储

## 🆘 常见问题

### Q1: 审核需要多长时间？
**A:** 通常1-7个工作日，周末和节假日可能延长。

### Q2: 审核被驳回怎么办？
**A:**
1. 查看驳回原因
2. 根据提示修改问题
3. 重新提交审核
4. 重新审核仍需1-7个工作日

### Q3: 如何快速通过审核？
**A:**
1. 确保功能完整，没有空页面
2. 页面截图真实反映功能
3. 内容符合平台规范
4. 用户体验良好
5. 选择合适的类目

### Q4: 发布后可以修改吗？
**A:** 可以。修改后上传新版本，重新审核和发布。

### Q5: 个人账号和企业账号有什么区别？
**A:**

| 功能 | 个人账号 | 企业账号 |
|-----|---------|---------|
| 注册费用 | 免费 | 300元/年 |
| 微信支付 | ❌ | ✅ |
| 微信登录 | ❌ | ✅ |
| 用户数量 | 无限制 | 无限制 |
| 支付功能 | 不支持 | 支持 |
| 开放接口 | 部分支持 | 完整支持 |

### Q6: 如何获取更多用户？
**A:**
1. 优化小程序标题和简介，便于搜索
2. 利用公众号推广
3. 添加好友分享功能
4. 定期更新内容和功能
5. 提供优质的服务和内容
