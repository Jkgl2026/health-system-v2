## 🌐 立即部署 Web 版

### 3步部署（5分钟）

1. **访问 Cloudflare Dashboard**
   - 网址：https://dash.cloudflare.com/
   - 登录或注册账号

2. **创建 Pages 项目**
   - Pages → Create a project
   - Connect to Git
   - 选择仓库：`Jkgl2026/health-system-v2`

3. **配置构建**
   ```
   Framework: Next.js (Static HTML Export)
   Build: npx next build
   Output: out
   ```
   - 点击 Save and Deploy
   - 等待 1-2 分钟

**完成！** 你的 Web 版已经上线了！

## 📱 立即部署小程序版

### 步骤 1：注册小程序（5分钟）

1. 访问 https://mp.weixin.qq.com/
2. 注册账号（**个人免费**）
3. 获取 AppID

**费用：** ¥0 免费

---

### 步骤 2：安装开发工具（2分钟）

1. 下载微信开发者工具
   - 网址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
2. 安装并登录

---

### 步骤 3：打开项目（2分钟）

1. 打开微信开发者工具
2. 点击 **导入项目**
3. 选择目录：`miniprogram/health-miniprogram/dist`
4. 填写 AppID
5. 点击 **导入**

**完成！** 小程序已在开发工具中运行。

---

### 步骤 4：上传代码（2分钟）

1. 点击工具栏的 **上传**
2. 填写版本号：`1.0.0`
3. 填写备注：`首次发布`
4. 点击 **上传**

---

### 步骤 5：提交审核（5分钟）

1. 访问 https://mp.weixin.qq.com/
2. 版本管理 → 开发版本
3. 选择刚才上传的版本
4. 点击 **提交审核**
5. 填写信息：
   - 类目：`生活服务 > 生活服务`
   - 功能页面：首页

6. 等待审核（1-7个工作日）

---

### 步骤 6：发布上线

审核通过后，点击 **发布** 即可。

## 📞 需要帮助？

### 常见问题

**Q: Web 版部署后打不开？**
A: 检查 Cloudflare Pages 构建日志，确保构建成功。

**Q: 小程序审核被拒？**
A: 选择"生活服务"类目，完善所有功能页面。

**Q: 如何修改症状列表？**
A:
- Web 版：编辑 `src/app/check/page.tsx`
- 小程序版：编辑 `miniprogram/health-miniprogram/src/pages/check/index.tsx`

**Q: 如何备份数据？**
A:
- Web 版：导出 localStorage 数据
- 小程序版：使用 `miniDataManager.exportData()`
