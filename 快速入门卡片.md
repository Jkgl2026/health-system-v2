## 🎯 评分标准（看懂数据）

### 健康评分（满分100）
```
80-100分：优秀 ✅
60-79分：良好 ✅
40-59分：一般 ⚠️
0-39分：较差 ❌
```

### BMI（身体质量指数）
```
<18.5：体重过轻
18.5-24：体重正常 ✅
24-28：超重
>28：肥胖
```

### 血压
```
<120/80：正常 ✅
120-130/80：正常高值
130-140/80-90：高血压1期
>140/90：高血压2期
```
