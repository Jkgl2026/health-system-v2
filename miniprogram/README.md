# 健康自检小程序

基于原Web版健康自检应用复刻的微信小程序版本。

## 项目结构

```
miniprogram/
├── app.js              # 小程序入口文件
├── app.json            # 小程序配置文件
├── app.wxss            # 全局样式
├── project.config.json # 项目配置
├── sitemap.json        # 索引配置
├── utils/              # 工具类
│   ├── api.js          # API请求封装
│   ├── storage.js      # 本地存储管理
│   └── health-data.js  # 健康数据常量
└── pages/              # 页面文件
    ├── index/          # 首页
    ├── personal-info/  # 个人信息页
    ├── check/          # 身体语言简表页
    ├── habits/         # 不良生活习惯页
    ├── symptoms300/    # 300症状表页
    ├── target/         # 重点症状选择页
    ├── choices/        # 三个选择页
    ├── requirements/   # 四个要求页
    ├── health-result/  # 健康分析结果页
    ├── my-solution/    # 我的方案页
    ├── story/          # 系统战役故事页
    ├── seven-questions/# 健康七问页
    └── admin/          # 后台管理
        ├── login/      # 登录页
        ├── dashboard/  # 仪表盘页
        └── user-detail/# 用户详情页
```

## 功能特性

### 前端功能
1. **首页** - 健康数据概览、功能介绍、自检流程说明
2. **个人信息页** - 姓名、手机号、年龄、性别、身高、体重录入
3. **身体语言简表** - 100项症状分类展示、搜索、选择
4. **不良生活习惯** - 252项习惯分类展示、搜索、选择
5. **300症状表** - 300项症状列表、搜索、选择
6. **重点症状选择** - 从已选症状中选择3个重点
7. **三个选择** - 健康意识、改变决心、方法路径的选择
8. **四个要求** - 坚持执行、按时复查、记录反馈、开放心态
9. **健康分析结果** - 健康评分、风险评估、中医分析、改善建议
10. **我的方案** - 个人健康方案展示
11. **系统战役故事** - 7章健康管理故事
12. **健康七问** - 7个健康相关问答

### 后台管理功能
1. **登录页** - 管理员登录（默认账号：admin / admin123）
2. **仪表盘** - 用户统计、检测统计、用户列表
3. **用户详情** - 用户信息查看、检测历史

## 使用说明

### 导入项目
1. 下载微信开发者工具
2. 导入`miniprogram`目录
3. 填写AppID（或使用测试号）

### 功能测试
1. 首页点击"开始自检"或"演示模式"
2. 按流程填写个人信息、选择症状和习惯
3. 完成三个选择和四个要求
4. 查看健康分析结果和个人方案

### 后台管理
1. 点击底部"后台"进入登录页
2. 使用默认账号登录：admin / admin123
3. 查看用户数据和统计信息

## 技术栈

- 微信小程序原生框架
- WXML / WXSS / JavaScript
- 本地存储 wx.setStorageSync

## 注意事项

1. 本项目为前端静态版本，数据存储在本地
2. 如需接入后端API，请修改`utils/api.js`
3. 健康数据常量定义在`utils/health-data.js`
