## 二、快速开始

### 步骤1：查看交付清单

打开 `健康自检后台系统-完整交付清单.md`，了解所有交付物。

### 步骤2：阅读部署手册

打开 `健康自检后台系统-部署操作手册.md`，按照手册完成环境搭建和部署。

### 步骤3：恢复数据库

```bash
# 1. 创建数据库
createdb health_check_db

# 2. 恢复结构
psql health_check_db < health_check_db_schema.sql

# 3. 恢复数据
psql health_check_db < health_check_db_data.sql
```

### 步骤4：启动项目

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm run dev

# 或构建并启动生产服务器
pnpm run build
pnpm run start
```

### 步骤5：访问系统

- 地址：http://localhost:5000
- 账号：admin

## 五、技术支持

### 常见问题

**Q1：数据库连接失败怎么办？**
A：检查 `src/lib/db.ts` 中的数据库配置是否正确，确保PostgreSQL服务正在运行。

**Q2：端口5000被占用怎么办？**
A：修改 `next.config.js` 中的端口号，或杀死占用5000端口的进程。

**Q3：依赖安装失败怎么办？**
A：尝试使用 `npm install` 或 `yarn install` 替代 `pnpm install`。

**Q4：如何添加新的管理员账号？**
A：直接在数据库 `sys_admin` 表中插入新记录。
