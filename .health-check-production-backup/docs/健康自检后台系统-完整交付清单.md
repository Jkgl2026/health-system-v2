## 四、部署操作手册

### 4.1 环境依赖清单
- Node.js 24+
- PostgreSQL 14+
- pnpm (包管理器)
- Nginx (Web服务器，可选)

### 4.2 本地开发启动
1. 安装依赖：`pnpm install`
2. 配置数据库：修改 `src/lib/db.ts` 中的数据库连接信息
3. 启动开发服务器：`pnpm run dev`
4. 访问：http://localhost:5000

### 4.3 生产环境部署
1. 安装依赖：`pnpm install`
2. 构建项目：`pnpm run build`
3. 启动生产服务器：`pnpm run start`
4. 配置Nginx反向代理
5. 配置SSL证书（可选）

### 4.4 数据库恢复
1. 创建数据库：`createdb health_check_db`
2. 恢复结构：`psql health_check_db < health_check_db_schema.sql`
3. 恢复数据：`psql health_check_db < health_check_db_data.sql`
