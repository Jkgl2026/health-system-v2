### 5. 健康分析自动生成逻辑（待实现）

**要求**：
系统必须根据用户填写内容自动生成专业分析，至少包含：
1. 综合健康评价
2. 风险等级与预警
3. 异常指标提示
4. 生活习惯改善建议
5. 睡眠质量分析
6. 压力与心理状态分析
7. 运动与饮食指导
8. 复查/就医提醒
9. 多次自检趋势变化
10. 个性化健康方案

**实现位置**：创建 src/lib/healthAnalyzer.ts

**代码框架**：
```typescript
export function generateHealthAnalysis(userData: any): string {
  const analysis: string[] = [];

  // 1. 综合健康评价
  analysis.push(`【综合健康评价】\\n`);
  analysis.push(`您的综合健康评分为${userData.health_score}分，健康状态为"${userData.health_status}"。`);

  // 2. 风险等级与预警
  analysis.push(`\\n【风险等级与预警】\\n`);
  if (userData.health_score >= 80) {
    analysis.push(`您的健康状况良好，继续保持健康的生活方式。`);
  } else if (userData.health_score >= 60) {
    analysis.push(`您的健康状况一般，建议关注以下几个方面...`);
  } else {
    analysis.push(`您的健康状况需要重点关注，强烈建议您...`);
  }

  // 3. 异常指标提示
  analysis.push(`\\n【异常指标提示】\\n`);
  if (userData.score_life < 60) {
    analysis.push(`生活方式得分偏低（${userData.score_life}分），需要改善...`);
  }
  // ... 其他指标

  // 4. 生活习惯改善建议
  // 5. 睡眠质量分析
  // 6. 压力与心理状态分析
  // 7. 运动与饮食指导
  // 8. 复查/就医提醒
  // 9. 多次自检趋势变化
  // 10. 个性化健康方案

  return analysis.join('\\n');
}
```

## 🔧 Coze对接说明

### 对接接口
1. POST /api/user/add - 新增用户（Coze专用）
2. POST /api/user/update - 更新健康数据

### 数据格式
```json
{
  "name": "张三",
  "phone": "13800138000",
  "age": 30,
  "gender": "男",
  "height": 175,
  "weight": 70,
  "job": "软件工程师",
  "sleep": "良好",
  "drink_smoke": "不抽烟、不喝酒",
  "exercise": "每周3次",
  "allergy": "无",
  "sickness": "无",
  "symptom": "偶尔头痛",
  "complete": 80,
  "health_status": "良好",
  "health_score": 75,
  "score_life": 70,
  "score_sleep": 80,
  "score_stress": 75,
  "score_body": 72,
  "score_risk": 78,
  "done_self_check": true,
  "done_require": false,
  "answer_content": "{\"question1\": \"答案1\", \"question2\": \"答案2\"}",
  "analysis": "健康分析内容..."
}
```

### 对接规则
1. 用户完成自检 → 调用 /user/add 或 /user/update
2. 所有字段必须完整传入，不得缺失
3. answer_content 存储**原始题目+答案JSON**
4. analysis 存储**完整健康分析文本**
5. 后台必须实时读取、原样展示、不截断、不丢失
6. Coze 提交 → 后台立即显示，无延迟
