# 快速开始：绑定自己的域名（简化版）

## 3 分钟快速了解

### 1. 如何拥有自己的域名

**最简单的方式：**
- 去阿里云或腾讯云购买
- 搜索喜欢的名字（如 `myhealth.com`）
- 支付 69-100元/年
- 完成

**推荐域名服务商：**
| 服务商 | 价格 | 特点 |
|--------|------|------|
| 阿里云 | 69元/年 | 国内访问快 |
| 腾讯云 | 69元/年 | 国内访问快 |
| Namecheap | $8-10/年 | 国际知名 |

### 2. 如何绑定域名（5 步完成）

**第1步：注册 Cloudflare（免费）**
```
1. 访问 https://dash.cloudflare.com/sign-up
2. 填写邮箱、密码
3. 验证邮箱
```

**第2步：添加域名**
```
1. 登录 Cloudflare
2. 点击 "Add a site"
3. 输入你的域名（如 myhealth.com）
4. 选择 Free（免费计划）
```

**第3步：修改 DNS 服务器**
```
Cloudflare 会提供两个服务器，类似：
- dina.ns.cloudflare.com
- eric.ns.cloudflare.com

去你的域名服务商（阿里云/腾讯云）：
1. 找到域名管理
2. 修改 DNS 服务器
3. 填入 Cloudflare 的两个服务器
4. 保存，等待生效（10分钟-24小时）
```

**第4步：创建隧道**
```
1. 在 Cloudflare，点击左侧菜单 Zero Trust → Tunnels
2. 点击 "Create a tunnel"
3. 命名（如 health-system-tunnel）
4. 保存
5. 点击 "Configure" → 选择你的域名 → 添加 DNS 记录
6. 保存
```

**第5步：启动隧道**
```bash
# 在服务器上执行
cloudflared tunnel --config /etc/cloudflared/config.yml run

# 或者使用 Token
cloudflared tunnel run --token <你的Token>
```

**完成！现在可以访问：**
- https://myhealth.com/admin/login
- https://www.myhealth.com/admin/dashboard

## 完整示例（以阿里云为例）

### 购买域名
```
网站：https://wanwang.aliyun.com/
搜索：myhealth.com
价格：69元/年
支付：支付宝/微信
```

### 添加到 Cloudflare
```
1. 登录 https://dash.cloudflare.com/
2. 添加站点：myhealth.com
3. 选择 Free
4. 记录 Cloudflare 提供的 DNS 服务器
```

### 修改 DNS
```
1. 登录阿里云
2. 域名控制台 → 域名管理
3. DNS 修改 → 修改 DNS 服务器
4. 填入 Cloudflare 的服务器
5. 等待生效
```

### 创建隧道
```
1. Cloudflare → Zero Trust → Tunnels
2. Create tunnel → 命名 → 保存
3. Configure → Add domain → 保存
```

### 启动服务
```bash
cloudflared tunnel run --token <你的Token>
```

### 访问
```
https://myhealth.com/admin/login
用户名：admin
密码：123456
```

## 常见问题（FAQ）

**Q：需要多少钱？**
A：
- 域名：69-100元/年
- Cloudflare：免费
- 合计：69-100元/年

**Q：需要多久？**
A：
- 购买域名：5分钟
- 添加到 Cloudflare：5分钟
- DNS 生效：10分钟 - 24小时
- 总计：首次配置 1小时左右

**Q：难吗？**
A：不太难，跟着文档一步步来就行。

**Q：可以绑定多个域名吗？**
A：可以，每个域名都需要添加到 Cloudflare。

**Q：Quick Tunnel 和命名隧道有什么区别？**
A：
- Quick Tunnel：随机域名，免费，简单
- 命名隧道：自己的域名，免费，需要配置

**Q：域名到期了怎么办？**
A：及时续费，每年 69-100元。
