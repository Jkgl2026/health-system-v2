## 📱 如何获取您的局域网 IP

### Windows 系统

#### 方法 1：使用 ipconfig
1. 按 `Win + R`，输入 `cmd`，回车
2. 输入命令：
   ```cmd
   ipconfig
   ```
3. 找到 **IPv4 地址**，类似：
   ```
   IPv4 地址 . . . . . . . . . . . : 192.168.1.100
   ```

#### 方法 2：使用设置界面
1. 打开 **设置** → **网络和 Internet**
2. 点击 **属性**
3. 查看IPv4 地址

### macOS 系统

#### 方法 1：使用系统偏好设置
1. 打开 **系统偏好设置** → **网络**
2. 选择当前网络连接
3. 查看 **IP 地址**

#### 方法 2：使用终端
```bash
ifconfig
```

### Linux 系统

```bash
ip addr show
```

或

```bash
hostname -I
```

## 🚫 如果无法访问，怎么办？

### 问题 1：防火墙阻止

#### Windows 防火墙配置
1. 打开 **Windows Defender 防火墙**
2. 点击 **高级设置**
3. 选择 **入站规则** → **新建规则**
4. 选择 **端口** → **TCP**
5. 特定本地端口：`5000`
6. 选择 **允许连接**
7. 全部选中（域、专用、公用）
8. 名称：`Health Check System`

#### Linux 防火墙配置
```bash
# 允许 5000 端口
sudo ufw allow 5000/tcp

# 查看防火墙状态
sudo ufw status

# 如果防火墙未启用，启用它
sudo ufw enable
```

### 问题 2：路由器防火墙

某些路由器有内置防火墙，可能阻止局域网设备互相访问。

#### 解决方法：
1. 登录路由器管理界面（通常是 `192.168.1.1` 或 `192.168.0.1`）
2. 查找 **防火墙设置** 或 **安全设置**
3. 启用 **局域网设备互通** 或 **AP 隔离** 设置

### 问题 3：VPN 影响

如果您使用了 VPN，可能影响局域网访问。

#### 解决方法：
- 暂时关闭 VPN
- 或在 VPN 设置中启用 **局域网访问**

### 问题 4：IP 地址变化

动态 IP 地址可能会变化。

#### 解决方法：
- 将电脑设置为静态 IP
- 或定期检查 IP 地址

## 🌐 内网穿透方案（外网访问）

### 方案 1：ngrok（最简单）

#### 安装 ngrok
1. 访问 https://ngrok.com/
2. 注册账号
3. 下载 ngrok
4. 解压到任意目录

#### 使用 ngrok
```bash
# 进入 ngrok 目录
cd C:\path\to\ngrok

# 启动 ngrok
ngrok http 5000
```

#### 访问地址
ngrok 会输出一个公网地址：
```
Forwarding    https://xxxx-xx-xx-xx-xx.ngrok-free.app -> http://localhost:5000
```

在浏览器访问：
```
https://xxxx-xx-xx-xx-xx.ngrok-free.app/admin/login
```

### 方案 2：frp（稳定，适合长期使用）

#### frp 客户端配置

创建 `frpc.ini` 配置文件：
```ini
[common]
server_addr = <frp服务器地址>
server_port = 7000
token = <您的token>

[health_check_system]
type = http
local_ip = 127.0.0.1
local_port = 5000
custom_domains = your-domain.com
```

#### 启动 frp 客户端
```bash
./frpc -c frpc.ini
```

## 📞 常见问题

### Q1: 为什么手机无法访问？
A: 检查以下几点：
- 手机和电脑是否连接到同一 WiFi
- 防火墙是否阻止 5000 端口
- 路由器是否启用了 AP 隔离

### Q2: 如何设置静态 IP？
A: 在网络设置中，选择 **手动配置**，设置：
- IP 地址：192.168.1.100
- 子网掩码：255.255.255.0
- 网关：192.168.1.1
- DNS：192.168.1.1

### Q3: ngrok 免费版有什么限制？
A: ngrok 免费版：
- 随机域名
- 每次重启会变化
- 有流量限制

### Q4: 如何让家人也能访问？
A: 有两种方式：
1. **局域网访问**：家人连接到同一 WiFi，使用局域网 IP 访问
2. **公网访问**：使用内网穿透工具（ngrok、frp）
