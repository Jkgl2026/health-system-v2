## 📱 安装步骤

### ⚠️ 重要安装提示

在开始安装之前，请务必注意以下几点：

1. **确保在应用首页安装**
   - 必须在应用首页看到"健康自我管理"标题和内容时，才能点击安装按钮
   - 不要在登录页面或空白页面安装

2. **不要在登录页面安装**
   - 如果看到"欢迎使用扣子"、"手机号登录"或"账号登录"等登录页面
   - 请先完成登录或刷新页面，直到看到应用内容再安装

3. **刷新页面重新安装**
   - 如果已经安装但打开后显示登录页面
   - 请删除已安装的应用，刷新页面后重新安装

### 1️⃣ Android 手机（最简单）

**自动安装方式**：
1. 使用 Chrome 浏览器打开应用
2. **确保看到应用内容**：确认页面显示"健康自我管理"标题，不是登录页面
3. 等待 3 秒，屏幕下方会自动弹出安装提示
4. 点击「安装应用到桌面」
5. 确认安装，完成！

**手动安装方式**：
1. 打开应用首页（确保看到应用内容）
2. 点击浏览器右上角菜单（三个点）
3. 选择「安装应用」或「添加到主屏幕」
4. 点击「安装」

---

### 2️⃣ iPhone / iPad（iOS）

由于 iOS 系统限制，需要手动添加：

1. **必须使用 Safari 浏览器**打开应用
2. **确保看到应用内容**：确认页面显示"健康自我管理"标题，不是登录页面
3. 点击底部中间的「分享」按钮（⬆️ 箭头图标）
4. 在弹出的菜单中**向下滚动**
5. 找到并点击「添加到主屏幕」
6. 点击右上角的「添加」按钮
7. 完成！应用图标会出现在主屏幕

💡 **详细图文指导**：访问应用首页，点击「如何安装到桌面」查看完整步骤图解。

---

### 3️⃣ Windows 电脑

**Chrome 浏览器**：
1. 打开应用首页（确保看到应用内容）
2. 等待地址栏右侧显示安装图标
3. 点击安装图标，选择「安装应用」
4. 点击「安装」按钮
5. 完成！应用会在桌面和开始菜单中创建快捷方式

**Edge 浏览器**：
1. 打开应用首页（确保看到应用内容）
2. 点击浏览器菜单（三个点）
3. 选择「应用」→「将此站点作为应用安装」
4. 点击「允许」
5. 点击「安装」按钮

---

### 4️⃣ Mac 电脑

**Chrome 浏览器**：
1. 打开应用首页（确保看到应用内容）
2. 等待地址栏右侧显示安装图标
3. 点击图标，选择「安装应用」
4. 点击「安装」按钮
5. 完成！应用会在应用程序文件夹和桌面创建快捷方式

**Safari 浏览器**：
1. 打开应用首页（确保看到应用内容）
2. 点击菜单栏的「文件」
3. 选择「添加到 Dock」
4. 应用图标会出现在 Dock 栏中

## 🔧 常见问题

### Q: 为什么 iPhone 没有自动弹出安装提示？
**A:** iOS 系统为了保护用户隐私，不支持 PWA 自动安装。您必须使用 Safari 浏览器，通过分享按钮手动添加到主屏幕。

### Q: 安装后应用图标显示不完整？
**A:** 第一次打开应用时可能会显示加载动画。关闭应用后再次打开，图标就会正常显示了。

### Q: 如何卸载已安装的应用？
**A:** 与卸载普通应用相同：
- **手机**：长按图标，选择删除或移除
- **电脑**：右键点击图标，选择卸载或删除

**注意**：删除应用不会影响您在云端保存的健康数据。

### Q: 应用会占用多少手机存储空间？
**A:** PWA 应用几乎不占用存储空间！它只是一个网页快捷方式，所有数据都存储在云端服务器。

### Q: 需要手动更新应用吗？
**A:** 不需要！每次打开应用时都会自动获取最新版本。

### Q: 可以离线使用吗？
**A:** 部分功能支持离线访问，但完整的健康自检和分析功能需要网络连接。
