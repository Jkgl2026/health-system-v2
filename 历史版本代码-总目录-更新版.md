## 💡 技术要点总结

### 1. 服务器端渲染 (SSR) 注意事项
```typescript
// ❌ 错误：直接访问 localStorage
const data = localStorage.getItem('key');

// ✅ 正确：先检查环境
const data = typeof window !== 'undefined' 
  ? localStorage.getItem('key') 
  : null;

// ✅ 更好：使用封装的工具类
const data = FormDataManager.getSafe('key');
```

### 2. 多条件查询优化
```typescript
// 优先级：phoneGroupId > phone > name
let queryParam = '';
if (user.phoneGroupId) {
  queryParam = `phoneGroupId=${encodeURIComponent(user.phoneGroupId)}`;
} else if (user.phone) {
  queryParam = `phone=${encodeURIComponent(user.phone)}`;
} else {
  queryParam = `name=${encodeURIComponent(user.name)}`;
}
```

### 3. 图标导入规范
```typescript
// 导入前确认图标存在
import { Activity, Heart, Brain } from 'lucide-react';

// 使用替代方案
const icons = {
  heart: Heart,
  kidney: Activity,  // ✅ 使用可用的图标替代
};
```
