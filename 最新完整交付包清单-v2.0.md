### 5. 部署步骤

#### 步骤1：上传代码到服务器

```bash
# 使用scp上传
scp -r /local/path user@server:/remote/path

# 或使用git
git clone your-repo-url

```

#### 步骤2：安装依赖

```bash
cd /remote/path
pnpm install --production
```

#### 步骤3：构建项目

```bash
pnpm run build
```

#### 步骤4：配置Nginx（可选）

创建 `/etc/nginx/sites-available/health-check-admin`：

```nginx
upstream health_check_backend {
    server 127.0.0.1:5000;
    keepalive 64;
}

server {
    listen 80;
    server_name your-domain.com;

    location / {
        proxy_pass http://health_check_backend;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_cache_bypass $http_upgrade;
    }
}
```

启用配置：

```bash
sudo ln -s /etc/nginx/sites-available/health-check-admin /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl restart nginx
```

#### 步骤5：启动服务

**方式1：直接启动**

```bash
pnpm run start
```

**方式2：使用PM2（推荐）**

```bash
# 安装PM2
npm install -g pm2

# 启动服务
pm2 start npm --name "health-check-admin" -- start

# 查看日志
pm2 logs health-check-admin

# 查看状态
pm2 status

# 设置开机自启
pm2 startup
pm2 save
```

### 10. 备份与恢复

#### 数据库备份

```bash
# 备份数据库
pg_dump -h localhost -U postgres health_check_db > backup_$(date +%Y%m%d).sql

# 恢复数据库
psql -h localhost -U postgres health_check_db < backup_20250204.sql
```

#### 自动备份脚本

创建 `/usr/local/bin/backup_db.sh`：

```bash
#!/bin/bash
BACKUP_DIR="/backup/postgresql"
DATE=$(date +%Y%m%d_%H%M%S)
DB_NAME="health_check_db"

mkdir -p $BACKUP_DIR

pg_dump -h localhost -U postgres $DB_NAME > $BACKUP_DIR/${DB_NAME}_${DATE}.sql

find $BACKUP_DIR -name "*.sql" -mtime +30 -delete

echo "Backup completed: ${DB_NAME}_${DATE}.sql"
```

设置定时任务：

```bash
crontab -e

# 添加每天凌晨2点备份
0 2 * * * /usr/local/bin/backup_db.sh
```
