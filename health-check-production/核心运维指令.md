## 🚀 服务管理

### 启动服务
```bash
cd /workspace/projects/health-check-production
coze dev >/app/work/logs/bypass/dev.log 2>&1 &
```

### 停止服务
```bash
# 查找进程 ID
ps aux | grep next-server

# 停止进程
kill <PID>

# 或者强制停止
kill -9 <PID>
```

### 重启服务
```bash
# 停止服务
kill $(ps aux | grep next-server | grep -v grep | awk '{print $2}')

# 启动服务
cd /workspace/projects/health-check-production
coze dev >/app/work/logs/bypass/dev.log 2>&1 &
```

### 查看服务状态
```bash
# 检查端口监听
ss -lptn 'sport = :5000'

# 检查进程
ps aux | grep next-server

# 检查 HTTP 响应
curl -I http://localhost:5000
```

## 🔐 安全建议

### 1. 修改默认密码
```bash
su - postgres -c "psql -d health_check_db -c \"UPDATE sys_admin SET password = 'your_secure_password' WHERE username = 'admin';\""
```

### 2. 配置防火墙
```bash
# 仅开放必要端口
ufw allow 80/tcp
ufw allow 443/tcp
ufw enable
```

### 3. 配置 SSL 证书
```bash
# 使用 Certbot 安装 SSL
apt-get install -y certbot python3-certbot-nginx
certbot --nginx -d your-domain.com
```

### 4. 定期备份
```bash
# 创建备份脚本
cat > /root/backup.sh << 'EOF'
#!/bin/bash
BACKUP_DIR="/backups/health_check"
DATE=$(date +%Y%m%d_%H%M%S)
mkdir -p $BACKUP_DIR

# 备份数据库
su - postgres -c "pg_dump health_check_db > $BACKUP_DIR/health_check_db_$DATE.sql"

# 备份配置文件
cp /workspace/projects/health-check-production/.env $BACKUP_DIR/.env_$DATE

# 删除 7 天前的备份
find $BACKUP_DIR -name "*.sql" -mtime +7 -delete
EOF

chmod +x /root/backup.sh

# 添加到 crontab（每天凌晨 2 点备份）
echo "0 2 * * * /root/backup.sh" >> /etc/crontab
```
