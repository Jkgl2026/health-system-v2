- **操作系统**：Linux（推荐 Ubuntu 20.04+/CentOS 7+）

- **pnpm**：8+

### 2. 一键部署

```bash

tar -xzf health-check-system-v1.0.0.tar.gz
cd health-check-system

# 执行一键部署脚本
chmod +x scripts/deploy.sh
```

## 🔧 配置说明

### 环境变量

在项目根目录创建 `.env` 文件：

```bash
# 数据库连接配置
DATABASE_URL="postgresql://postgres:your_password@localhost:5432/health_check_db"

# 应用配置
NODE_ENV="production"
PORT=5000

# Coze SDK 配置（可选）
COZE_API_KEY="your_coze_api_key"
COZE_API_SECRET="your_coze_api_secret"
```

### 数据库配置

1. 创建数据库：
```bash
createdb health_check_db
```

2. 恢复数据库结构：
```bash
psql -d health_check_db -f database/health_check_db_schema.sql
```

3. 恢复数据库数据：
```bash
psql -d health_check_db -f database/health_check_db_data.sql
```

### 常见问题

#### 1. 数据库连接失败
检查 `src/lib/db.ts` 中的数据库连接信息

#### 2. 端口被占用
检查 5000 端口是否被占用，或修改 `PORT` 环境变量

#### 3. 依赖安装失败
尝试使用 `cnpm` 或 `npm` 替代 `pnpm`

#### 4. 页面404
检查路由配置和权限中间件
