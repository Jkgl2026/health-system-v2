### 方案 1：ngrok 免费版（推荐⭐⭐⭐⭐⭐）

#### 优势
- ✅ **无需任何密码**
- ✅ 完全免费
- ✅ 长期稳定
- ✅ 配置简单
- ✅ 自动 HTTPS

#### 配置步骤

##### 步骤 1：注册 ngrok 账户
1. 访问：https://ngrok.com/signup
2. 使用邮箱注册（免费）
3. 登录后会看到 Dashboard

##### 步骤 2：获取 Authtoken
1. 在 ngrok Dashboard 中
2. 进入 "Your Authtoken" 页面
3. 复制显示的 authtoken

##### 步骤 3：安装 ngrok（在沙箱中）
```bash
# 下载 ngrok
curl -s https://ngrok-agent.s3.amazonaws.com/ngrok.asc | sudo tee /etc/apt/trusted.gpg.d/ngrok.asc >/dev/null
echo "deb https://ngrok-agent.s3.amazonaws.com buster main" | sudo tee /etc/apt/sources.list.d/ngrok.list
sudo apt update && sudo apt install ngrok

# 验证安装
ngrok version
```

##### 步骤 4：配置 authtoken
```bash
# 替换成你的 authtoken
ngrok config add-authtoken <你的authtoken>
```

##### 步骤 5：启动隧道
```bash
# 启动隧道（会在后台运行）
ngrok http 5000 --log=stdout &
```

##### 步骤 6：获取访问地址
启动后会显示：
```
Forwarding https://xxxx-xx-xx-xx-xx.ngrok-free.app -> http://localhost:5000
```

这个地址就是你的访问地址，例如：
```
https://abc-def-123-456.ngrok-free.app/admin/dashboard
```

##### 步骤 7：设置开机自启
创建 systemd 服务：
```bash
sudo tee /etc/systemd/system/ngrok.service > /dev/null <<EOF
[Unit]
Description=ngrok tunnel
After=network.target

[Service]
Type=simple
User=root
ExecStart=/usr/bin/ngrok http 5000
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
EOF

# 启动服务
sudo systemctl daemon-reload
sudo systemctl start ngrok
sudo systemctl enable ngrok
```
