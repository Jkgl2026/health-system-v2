## 三、完整测试流程

### 步骤1：用户端测试
```
1. 启动小程序
2. 进入首页，查看评分和快速入口
3. 进入自检页面，选择症状并生成报告
4. 查看结果页面，确认评分和建议
5. 进入分析页面，查看趋势图和记录
6. 进入我的页面，登录微信账号
7. 测试数据导出和导入功能
```

### 步骤2：管理员后台测试
```
1. 访问管理后台登录页（pages/admin/login/index）
2. 使用默认账号登录（admin / admin123）
3. 查看管理后台首页的数据统计
4. 点击「用户管理」，查看用户列表
5. 测试用户搜索和详情查看
6. 点击「记录管理」，查看记录列表
7. 测试筛选和导出功能
8. 测试退出登录
```

### 步骤3：联动测试
```
1. 在用户端完成一次新的自检
2. 切换到管理后台首页
3. 点击刷新，确认统计数据更新
4. 进入记录管理页，确认新记录显示
5. 测试筛选和详情查看
```

## 八、已知限制

1. **用户端数据存储**：使用本地 Storage，数据仅保存在当前设备
2. **管理员登录**：使用固定账号密码，生产环境需要改为动态验证
3. **文件导入导出**：需要在真机上测试，模拟器可能受限
4. **IP 获取**：小程序无法获取真实 IP，登录日志显示 'unknown'
5. **数据同步**：用户端和管理员端共享同一 Storage，数据实时同步
