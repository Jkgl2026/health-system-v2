# 健康自检小程序

一个基于 Taro 3.6.x + React + TypeScript 开发的微信健康自检小程序，帮助用户了解自身健康状况。

## 📱 功能特性

### 核心功能
- ✅ **症状自检**：选择症状，获取健康评分和等级
- ✅ **结果分析**：展示评分、等级、各维度得分、健康建议
- ✅ **数据统计**：查看自检次数、平均分、最高/最低分
- ✅ **趋势分析**：Canvas 折线图展示健康趋势变化
- ✅ **历史记录**：查看历史自检记录，支持点击查看详情
- ✅ **健康贴士**：提供实用的健康建议

### 用户功能
- ✅ **微信登录**：支持微信授权登录，保存用户信息
- ✅ **数据导出**：将自检记录导出为 JSON 文件
- ✅ **数据导入**：从 JSON 文件导入历史数据
- ✅ **数据清空**：一键清除所有自检记录
- ✅ **退出登录**：清除用户登录状态

### 管理员功能
- ✅ **登录验证**：安全的账号密码登录
- ✅ **数据统计**：用户数、自检次数、平均分、高风险用户
- ✅ **等级分布**：Canvas 环形图展示健康等级分布
- ✅ **用户管理**：查看用户列表、搜索用户、查看用户详情
- ✅ **记录管理**：查看所有自检记录、按条件筛选、查看详情
- ✅ **数据导出**：导出用户数据和记录数据
- ✅ **权限控制**：所有管理页面需要登录

## 🛠️ 技术栈

- **框架**：Taro 3.6.x

- **语言**：TypeScript 4.1.0

## 📊 健康评分体系

### 评分规则
- 基础分：100 分
- 根据选择的症状扣分
- 每个症状有对应的扣分值

### 健康等级

| 等级 | 分数范围 | 说明 |
|------|---------|------|
| 优秀 | 90-100 | 身体状况良好 |
| 良好 | 80-89 | 身体状况较好 |
| 一般 | 70-79 | 身体状况一般 |
| 需要关注 | 60-69 | 需要关注健康 |
| 建议就医 | 0-59 | 建议及时就医 |

### 症状分类

1. **头部**：头痛、头晕、耳鸣
2. **睡眠**：失眠、多梦、易醒
3. **消化**：食欲不振、胃痛、便秘
4. **体力**：乏力、气短、腰酸
5. **心理**：焦虑、抑郁、烦躁
6. **心脏**：心悸、胸闷、心慌

## 🐛 常见问题

### 1. 登录失败
**原因**：微信开发者工具未配置 AppID  
**解决**：在 project.config.json 中配置正确的 AppID

### 2. 导出/导入失败
**原因**：微信小程序文件系统权限限制  
**解决**：确保使用真机调试，开发工具可能有限制

### 3. 图表不显示
**原因**：Canvas 组件未正确初始化  
**解决**：检查 Canvas ID 是否唯一，确保数据正确传递

### 4. 数据丢失
**原因**：用户清除了小程序缓存  
**解决**：定期使用导出功能备份数据
