# 快速开始指南

## 第一步：安装依赖

```bash
cd miniprogram-code
npm install
```

## 第二步：配置小程序 AppID

打开 `project.config.json`，修改 `appid`：

```json
{
  "appid": "你的微信小程序AppID",
  "projectname": "健康自检小程序"
}
```

**如何获取 AppID**：
1. 访问 [微信公众平台](https://mp.weixin.qq.com/)
2. 注册并登录
3. 在「开发」->「开发设置」中查看 AppID

## 第三步：启动开发

```bash
npm run dev:weapp
```

## 第四步：打开微信开发者工具

1. 打开微信开发者工具
2. 导入项目，选择 `dist` 目录
3. 项目名称：健康自检小程序
4. AppID：使用配置的 AppID
5. 点击「导入」

## 第五步：开始使用

### 测试自检功能

1. 点击底部「自检」标签
2. 选择任意症状（如：头痛、失眠）
3. 点击「生成健康报告」
4. 查看评分和健康建议

### 测试登录功能

1. 点击底部「我的」标签
2. 点击「微信登录」按钮
3. 授权获取用户信息
4. 登录成功后查看用户信息

### 测试数据导出

1. 进入「我的」页面
2. 点击「导出数据」
3. 文件自动保存到手机

### 测试数据导入

1. 进入「我的」页面
2. 点击「导入数据」
3. 选择之前导出的 JSON 文件
4. 确认导入

## 常用命令

```bash
# 开发模式
npm run dev:weapp

# 构建生产版本
npm run build:weapp

# 清理构建文件
npm run clean
```

## 项目结构

```
miniprogram-code/
├── src/           # 源代码
│   ├── pages/     # 页面
│   ├── types/     # 类型定义
│   └── utils/     # 工具函数
├── config/        # 配置文件
├── dist/          # 构建输出
└── package.json   # 项目依赖
```

## 注意事项

1. **首次运行**：需要先运行 `npm install` 安装依赖
2. **修改代码**：修改后自动重新编译，刷新开发者工具即可
3. **真机调试**：建议使用真机调试，部分功能（如文件导入导出）在模拟器可能受限
4. **AppID**：测试时可以使用「测试号」，正式发布需要注册小程序
