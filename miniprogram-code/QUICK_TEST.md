# 快速测试指南

## 🚀 快速开始

### 1. 启动项目
```bash
cd miniprogram-code
npm install
npm run dev:weapp
```

### 2. 访问页面
打开微信开发者工具，扫描二维码或直接加载项目。

---

## 📱 用户端测试（5分钟）

### 测试1：完成一次健康自检
```
1. 点击「自检」
2. 选择 3-5 个症状（如：头痛、乏力、失眠）
3. 点击「生成健康报告」
4. 查看评分和健康建议
5. 点击「返回首页」
```
✅ 验证：评分正确，建议合理

### 测试2：查看健康趋势
```
1. 点击「健康分析」
2. 查看统计数据和趋势图
3. 点击记录查看详情
4. 查看健康贴士
```
✅ 验证：图表正常，数据准确

### 测试3：登录和数据管理
```
1. 点击「我的」
2. 点击「微信登录」并授权
3. 查看用户信息
4. 点击「导出数据」
5. 返回首页再次自检
6. 再次导出数据
```
✅ 验证：登录正常，数据导出成功


## 🐛 常见问题

### 问题1：登录失败
**解决**：检查账号密码是否正确（admin / admin123）

### 问题2：图表不显示
**解决**：刷新页面或重新编译

### 问题3：数据导出失败
**解决**：确保有数据可导出，或使用真机测试

### 问题4：页面空白
**解决**：清除缓存，重新编译
