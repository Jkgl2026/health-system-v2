## 🚀 快速开始

### 安装依赖
```bash
cd miniprogram-code
npm install
```

### 启动开发
```bash
npm run dev:weapp
```

### 访问管理后台
1. 在开发者工具访问：`pages/admin/login/index`
2. 账号：`admin`
3. 密码：`admin123`

---

## ⚙️ 配置说明

### 微信小程序配置
修改 `project.config.json` 中的 `appid`：

```json
{
  "appid": "你的微信小程序AppID"
}
```

### 管理员账号密码
修改 `src/utils/adminAuth.ts`：

```typescript
private static readonly DEFAULT_USERNAME = 'admin'
private static readonly DEFAULT_PASSWORD = 'admin123'
```

## 📖 使用指南

### 用户端使用
1. 打开小程序
2. 点击「自检」选择症状
3. 查看健康评分和建议
4. 在「分析」页面查看趋势
5. 在「我的」页面管理数据

### 管理员使用
1. 登录管理后台
2. 查看数据统计
3. 管理用户和记录
4. 导出数据进行分析

---

## 🔒 安全说明

1. **默认账号**：生产环境必须修改默认账号密码
2. **数据隐私**：用户数据仅保存在本地
3. **权限控制**：所有管理员页面需要登录
4. **只读规则**：管理员后台不修改用户数据
5. **会话管理**：登录会话24小时过期

---

## 🐛 已知限制

1. 本地存储，数据不跨设备
2. 无法获取真实 IP 地址
3. 文件导入导出需真机测试
4. 管理员账号固定（可配置）
5. 无云端数据同步
