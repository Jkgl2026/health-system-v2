# 后台用户详细内容横向布局设计

## 设计目标

重新设计后台用户详细内容，采用横向布局（类似电脑屏幕），使用4*4、8*8、3*3网格系统，实现信息密集、直观易用的界面。

## 横向布局实现

### 方案一：使用横向滚动
```tsx
<div className="overflow-x-auto">
  <div className="flex gap-4 min-w-max">
    {/* 所有模块 */}
  </div>
</div>
```

### 方案二：使用网格系统
```tsx
<div className="grid grid-cols-12 gap-4">
  <div className="col-span-3">基本信息(4×4)</div>
  <div className="col-span-3">综合健康评分(4×4)</div>
  <div className="col-span-3">健康状况全面解析(4×4)</div>
  <div className="col-span-3">健康改善路径(4×4)</div>
</div>
```

### 方案三：混合布局
```tsx
<div className="space-y-4">
  {/* 第一行：4个4×4模块 */}
  <div className="grid grid-cols-4 gap-4">
    <div className="col-span-1">基本信息</div>
    <div className="col-span-1">综合健康评分</div>
    <div className="col-span-1">健康状况全面解析</div>
    <div className="col-span-1">健康改善路径</div>
  </div>
  {/* 其他行... */}
</div>
```

## 视觉设计特点

### 1. 横向布局
- 宽屏设计：使用max-w-screen-2xl或更大的宽度
- 横向排列：信息从左到右排列
- 最小化垂直滚动

### 2. 网格系统
- 4×4网格：grid-cols-4，每个模块占1/4宽度
- 8×8网格：grid-cols-8，每个项目占1/8宽度
- 3×3网格：grid-cols-3，每个模块占1/3宽度

### 3. 卡片设计
- 统一的圆角和阴影
- 渐变背景区分不同类型
- 图标和标题清晰可见
- 悬停效果增强交互
