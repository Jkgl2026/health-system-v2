## 📝 注意事项

### 必须修改的配置
1. **project.config.json**
   ```json
   {
     "appid": "你的小程序AppID"
   }
   ```

2. **app.tsx**
   ```typescript
   wx.cloud.init({
     env: '你的云开发环境ID',
     traceUser: true,
   });
   ```

## 🆘 常见问题

### Q1: 如何获取小程序AppID？
A: 在微信公众平台（mp.weixin.qq.com）注册小程序，开通后即可获得AppID。

### Q2: 如何开通云开发？
A: 在微信开发者工具中，点击"云开发"按钮，根据提示开通。

### Q3: 如何创建云函数？
A: 
1. 在项目根目录创建 `cloudfunctions` 目录
2. 在目录中创建云函数文件夹（如 `user`）
3. 右键点击文件夹，选择"创建云函数"
4. 编写云函数代码
5. 右键点击云函数文件夹，选择"上传并部署"

### Q4: 如何调试云函数？
A: 在云函数的编辑器中，点击"本地调试"或"云端调试"按钮。

### Q5: 如何导出数据？
A: 在管理员端的"数据概览"页面，点击"导出数据"按钮，或者在"用户管理"和"记录管理"页面点击导出按钮。
