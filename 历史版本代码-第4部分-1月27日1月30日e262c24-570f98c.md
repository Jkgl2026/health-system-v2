## 📅 版本e262c24完整代码

### 版本信息
**提交ID**：e262c24  
**提交时间**：2026-01-28 11:42:19  
**提交信息**：fix: 修复后台历史记录查询功能，支持通过 phoneGroupId 和 name 查询

### 问题分析

#### 根本原因
1. 所有用户的手机号字段（`phone`）都是 `null`，导致基于手机号的历史记录查询失败
2. 原始代码只通过 `phone` 查询历史记录，当用户没有填写手机号时无法查询
3. 按钮显示条件只检查 `phone` 字段，导致大多数用户看不到"历史记录"按钮

#### 数据验证
通过调试 API 确认：
- 数据库中有 40 条用户记录
- 所有用户的 `phone` 字段都是 `null`
- 但所有用户都有 `name` 字段
- 部分用户有 `phoneGroupId` 字段

### 修复内容

#### 1. 修改历史记录查询逻辑
修改 `handleViewHistory` 函数，支持多种查询方式：
- 优先使用 `phoneGroupId` 查询
- 其次使用 `phone` 查询
- 最后使用 `name` 查询

```typescript
const handleViewHistory = async (user: any) => {
  setLoadingHistory(true);
  setHistoryPhone(user.phone || user.name);
  try {
    let queryParam = '';
    if (user.phoneGroupId) {
      queryParam = `phoneGroupId=${encodeURIComponent(user.phoneGroupId)}`;
    } else if (user.phone) {
      queryParam = `phone=${encodeURIComponent(user.phone)}`;
    } else {
      queryParam = `name=${encodeURIComponent(user.name)}`;
    }
    const response = await fetch(`/api/user/history?${queryParam}`);
    const data = await response.json();
    if (data.success) {
      setHistoryUsers(data.users);
    }
  } catch (error) {
    console.error('Failed to fetch history:', error);
  } finally {
    setLoadingHistory(false);
  }
};
```

#### 2. 修改按钮显示条件
从 `{userSummary.user.phone ? ...}` 改为 `{(userSummary.user.phone || userSummary.user.name) ? ...}`
确保用户有 `phoneGroupId` 或 `name` 时都能看到"历史记录"按钮

#### 3. 修改按钮调用
从 `onClick={() => handleViewHistory(userSummary.user.phone!)}`
改为 `onClick={() => handleViewHistory(userSummary.user)}`
传递完整的用户对象

#### 4. 更新类型定义
在 `UserSummary` 接口中添加 `phoneGroupId: string | null` 字段

```typescript
interface UserSummary {
  user: {
    id: string;
    name: string | null;
    phone: string | null;
    phoneGroupId: string | null;  // 新增
    email: string | null;
    age: number | null;
    gender: string | null;
    height: string | null;
    weight: string | null;
    occupation: string | null;
  };
  // ...其他字段
}
```

#### 5. 清理调试代码
删除调试 API 文件 `src/app/api/debug/check-phone-history/route.ts`
